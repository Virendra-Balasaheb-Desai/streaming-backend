# Video Streaming Backend API

A scalable backend for a **video streaming platform (similar to YouTube)** built using **Node.js, Express, and MongoDB**.
The project implements RESTful APIs for managing videos, comments, tweets, authentication, and file uploads.

## 🚀 Tech Stack

* **Backend Framework:** Node.js with Express
* **Database:** MongoDB
* **ODM:** Mongoose
* **Authentication:** JWT (Access Token + Refresh Token)
* **File Uploads:** Multer + Cloudinary
* **Password Hashing:** bcrypt
* **API Testing:** Postman

## ✨ Features

* Model–Controller architecture
* RESTful API design
* JWT authentication (Access + Refresh tokens)
* Secure password hashing with bcrypt
* File uploads to Cloudinary
* Comments and tweet functionality
* Aggregation pipelines and pagination
* Mongoose hooks (custom & built-in)
* Standard API response and error handling
* API testing with Postman

API testing with Postman

## ▶️ Getting Started

### 1. Clone the repository

```
git clone https://github.com/Virendra-Balasaheb-Desai/streaming-backend.git
cd streaming-backend
```

### 2. Install dependencies

```
npm install
```

### 3. Satisfy environment variables

Create a `.env` file using `.env.example`.

### 4. Run the server

```
npm run dev
```

or

```
npm start
```
